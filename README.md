# Interview Assistant App

Interview Assistant App is an AI-powered interview preparation tool built with Streamlit and the OpenAI API. It helps users prepare for job interviews by analyzing a CV, processing a job description, generating tailored interview questions, evaluating answers, and recommending learning resources.

## Overview

The application is designed to simulate a personalized interview preparation workflow. A user uploads a CV in PDF format, pastes a job description, receives 20 AI-generated interview questions, answers them, and gets structured feedback with improvement suggestions.

## Features

- Extracts text from uploaded CVs in PDF format
- Cleans and formats pasted job descriptions
- Generates 20 interview questions tailored to the candidate profile and job description
- Evaluates user answers with per-question feedback and an overall score
- Recommends learning resources based on weak areas
- Combines recommendations from a local resource library and AI-generated suggestions

## Installation

Create and activate a virtual environment, then install the required dependencies.

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Environment Variables

Create a `.env` file in the project root and add your OpenAI API key:

```env
OPENAI_API_KEY=your_api_key_here
```

## Running the Application

Start the Streamlit app with:

```bash
python -m streamlit run app.py
```

## Application Flow

1. The user uploads a CV in PDF format.
2. The application extracts and cleans the CV text.
3. The user pastes a job description.
4. The application cleans the job description text.
5. OpenAI generates 20 interview questions based on the CV and job description.
6. The user writes answers to the generated questions.
7. OpenAI evaluates the answers and returns scores and feedback.
8. The application recommends learning resources from the local library and AI suggestions.
